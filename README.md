# hotel-reservation-system
The Hotel Reservation System is a dynamic web application that simplifies the hotel booking process by allowing clients to create an account, securely log in, browse available rooms, make bookings, update their profiles, and examine payment details. Hotel managers can use an administrative dashboard to monitor room availability, reservations, client records, and payments, as well as generate reports to help with daily operations. The system stores and retrieves information dynamically using a MySQL database and PHP.
The technology is designed to replace manual reservation procedures with a more efficient and organized online platform. It allows clients to easily reserve hotel rooms while also assisting administrators in quickly managing bookings, monitoring room availability, documenting payments, and generating reports to enable better decision-making and hotel operations.
System Features
1. User Registration and Login
- Allows customers and administrators to create accounts and securely log in.User credentials are stored in a database for authentication.
2. Payment recording
- Records payment details such as amount paid, payment method, transaction date, and payment status. This helps the hotel track completed and pending payments.
3. Customer Profile Management
- Customers can update their personal information, such as name, contact number, email address, and password. The database stores all profile changes securely.
4. Admin Dashboard
- Gives administrators a centralized panel to manage reservations, view customer details, monitor room availability, and track payments.
5. Reporting and Analytics
- Generates reports on reservations, occupancy rates, and revenue. Helps hotel management make data-driven decisions and improve operations.
6. Room Availability Display
- Shows available rooms with type, price, and status. Data is retrieved from the database to reflect real-time availability.
